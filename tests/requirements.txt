# fakeredis stands in for Azure Cache for Redis in tests. The [lua] extra is
# REQUIRED, not optional: StateStore's dedup/threshold/storm windows are Lua

# EVAL and every state test fails.
#

# `EXPIRE key ttl NX`, which is Redis 7.0+ syntax that Azure Cache for Redis
# (6.0) rejects outright. Passing here is necessary, not sufficient; anything
# touching Redis command syntax deserves a check against a real 6.0 server
# (docker run redis:6.0) before it ships.
